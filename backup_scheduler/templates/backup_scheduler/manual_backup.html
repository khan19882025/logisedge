Backup Information
What is a Manual Backup?

A manual backup allows you to create a backup immediately without waiting for scheduled backups. This is useful for:

  • Before major system changes
  • After important data updates
  • Testing backup procedures
  • Emergency situations
Estimated Time

Backup time depends on data size and type:

  • Full Database: 10-30 minutes
  • Incremental: 5-15 minutes
  • Documents: 2-10 minutes
Safety Features

Your backup will be:

  • Encrypted for security
  • Verified for integrity
  • Logged for audit trail
  • Stored in secure location
