Import Instructions
Required Columns:
  • account_code - Unique account code (e.g., 1000, 1100)
  • name - Account name (e.g., Cash, Accounts Receivable)
  • account_type - Account type name (must exist in system)
  • parent_account - Parent account code (optional)
  • description - Account description (optional)
  • is_active - Active status (true/false, default: true)
Important Notes:
  • Account codes must be unique
  • Account types must already exist in the system
  • Parent account codes must exist if specified
  • First row should contain column headers
  • Use commas to separate values
