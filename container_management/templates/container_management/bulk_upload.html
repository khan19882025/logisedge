File Format Guide
Required Columns:
  • container_number - ISO 6346 standard container number
  • container_type - 20ft, 40ft, 40hc, reefer, open_top, flat_rack, tank
  • size - Container size (20ft, 40ft, etc.)
  • tare_weight - Empty container weight in kg
  • max_payload - Maximum payload in kg
  • status - available, booked, in_use, maintenance, retired
  • current_location - Current location (Port/Terminal/Yard)
Optional Columns:
  • yard_location - Specific yard location
  • line_operator - Shipping line operator
  • purchase_date - Purchase date (YYYY-MM-DD)
  • last_maintenance - Last maintenance date (YYYY-MM-DD)
  • next_maintenance - Next maintenance date (YYYY-MM-DD)
  • notes - Additional notes
