6. Django Template Usage

How to use in Django templates:

{% load static %} <!-- In your Django template --> <link rel="stylesheet" href="{% static 'css/currency_icons.css' %}"> <!-- Display currency with icon --> <div class="currency-with-icon"> <img src="{% static 'img/aed_currency.svg' %}" class="currency-icon currency-icon-aed" alt="AED"> <span>{{ amount }} {{ currency.code }}</span> </div> <!-- Or inline SVG --> <svg width="20" height="20" viewBox="0 0 24 24" class="currency-icon currency-icon-aed"> <!-- SVG content here --> </svg>

7. CSS Classes Available

  • .currency-icon - Base currency icon class
  • .currency-icon-aed - AED specific icon (20x20)
  • .currency-icon-aed-large - Large AED icon (32x32)
  • .currency-icon-aed-small - Small AED icon (16x16)
  • .currency-with-icon - Container for icon + text
  • .currency-selector - For currency selection dropdowns
  • .currency-column - For table currency columns
  • .currency-field - For form currency fields
