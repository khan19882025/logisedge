Naming Convention

Common Actions

  • create - Create new records
  • edit - Modify existing records
  • delete - Remove records
  • approve - Approval actions
  • export - Export data
Priority Levels
  • Critical - Security-sensitive
  • High - Business-critical
  • Medium - Standard access
  • Low - Basic access
